About Me

Hello! I'm a seasoned programmer with 25 years of coding experience. My journey began at the age of 12 when I started making games with RPG Maker 95 and RPG Maker 2k. This early passion for game development led me to learn HTML, which subsequently opened the doors to JavaScript. I quickly found myself diving into web development, creating games with a JavaScript engine called SphereJS, and expanding my skills in HTML, JS, and CSS through online learning.

Memorable Projects

One of my most memorable projects was creating a clothing shop for a personal client. The project featured a Rails backend with models for shirts, logos, and colors, as well as customizable T-shirts and clothing options. Despite it being outside my usual focus, I embraced the challenge and significantly improved my front-end skills, learning a lot about JavaScript for web design. This experience, along with other projects using Tailwind CSS, helped me discover a newfound passion for front-end development.
